# Voice Changer

## Steps to get your .exe (follow exactly):

1. Go to https://github.com/new
2. Repository name: `voice-changer`
3. Set to **Public**
4. Click **Create repository**
5. On the next page click **uploading an existing file**
6. Drag ALL files from this zip into the browser window
7. Click **Commit changes**
8. Click the **Actions** tab at the top
9. Click **Build Windows App** on the left
10. Click **Run workflow** → **Run workflow**
11. Wait 3 minutes for the green checkmark
12. Click the finished job → scroll to bottom → click **Voice-Changer-Setup**
13. Extract the downloaded zip → double-click the `.exe` → installed ✓
